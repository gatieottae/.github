# ✈️ 같이어때

> **“같이 여행 준비할래?”**  
> 그룹 여행을 위한 **올인원 협업 플랫폼**

---

## 🌍 서비스 소개

친구, 가족, 커플과 함께 여행을 준비할 때  
다음과 같은 문제를 자주 겪는다.

- 일정이 서로 맞지 않음
- 숙소·코스 선택에서 의견 충돌
- 비용 정산의 번거로움
- 단체 채팅방에서 정보가 흩어짐

**같이어때**는 이러한 문제를 해결하기 위해  
**일정 공유 · 실시간 투표 · 채팅 · 1/n 정산**을 하나로 제공하는  
**그룹 여행 협업 플랫폼**이다.

---

## 🎯 서비스 목적

- 여행 준비 과정의 **의사결정 스트레스 감소**
- 단체 채팅에 흩어진 정보를 **하나의 공간으로 통합**
- 여행 일정·비용·의견을 **투명하고 실시간으로 공유**

---

## 🔑 핵심 키워드

- 그룹 여행 일정 조율
- 실시간 투표 & 채팅
- 1/n 비용 정산 시뮬레이션
- 실시간 알림 센터 (Redis Pub/Sub + WebSocket)

---

## 👤 사용자 시나리오

| 시나리오 | 설명 |
|------|------|
| 그룹 만들기 | 사용자가 그룹을 생성하고 친구를 초대 |
| 일정 조율 | 각자 가능한 날짜를 등록하고 겹치는 날짜 확인 |
| 숙소/코스 투표 | 후보 항목에 대해 실시간 투표 |
| 채팅 소통 | 그룹 내 실시간 채팅으로 의견 조율 |
| 1/n 정산 | 총 금액 기준 자동 분담금 계산 |
| 알림 수신 | 투표 마감, 일정 확정, 결제 완료 알림 |

---

## 🧩 핵심 기능

| 기능 영역 | 세부 기능 | 설명 |
|------|------|------|
| 🧑‍🤝‍🧑 그룹 관리 | 그룹 생성 / 초대 / 참여 | 초대 코드 또는 링크 기반 |
| 📅 일정 캘린더 | 일정 후보 등록 / 겹침 표시 | 달력 UI, 실시간 동기화 |
| 🗳 투표 | 일정·숙소 투표 | 실시간 집계 및 마감 처리 |
| 💬 채팅 | 그룹 채팅 | WebSocket 기반, 시스템 메시지 포함 |
| 💸 1/n 계산기 | 비용 분담 | 금액 변경 시 실시간 반영 |
| 🔔 알림 센터 | 이벤트 알림 | Redis Pub/Sub → WebSocket |

## 🛠 기술 스택

| 영역 | 기술 |
|---|---|
| Frontend | React / TypeScript / Vite / Tailwind CSS |
| Backend | Spring Boot (Spring Web, Spring Data JPA) |
| Database | JPA / Hibernate / MySQL 또는 PostgreSQL |
| Realtime | Spring WebSocket / Redis Pub/Sub |
| Auth | JWT / Spring Security |
| Infra | Docker / Nginx / EC2 |
| Cache | Redis |
| Batch (Option) | Spring Batch (로그 → ES 적재) |

---

## 📦 Repositories

- **frontend** : 사용자 UI, 캘린더·채팅·투표 화면
- **backend** : API 서버, 실시간 처리
